# 隐私政策

**最后更新日期：2026年2月2日**

## 概述

MyIP 是一个开源的 macOS 菜单栏应用，用于显示您的网络状态和 IP 地址信息。我们非常重视您的隐私。

## 数据收集

MyIP **不收集、不存储、不传输**任何个人数据到远程服务器。

### 本地数据

应用仅在您的设备本地处理以下信息：

- **网络信息**：IP 地址、网关地址、网络接口信息
- **位置信息**（可选）：如果您启用位置服务，应用会获取您的大致地理位置用于显示在菜单中
- **设置信息**：应用偏好设置保存在本地文件系统

所有数据仅存储在您的 Mac 上，不会上传到任何服务器。

### 第三方服务

应用会访问以下第三方服务以获取网络信息：

1. **IP 地址查询服务**：
   - 用于获取您的外网 IP 地址
   - 仅发送 HTTP 请求，不包含任何个人身份信息

2. **高德地图 API**：
   - 用于查询 Direct IP 的地理位置
   - 仅发送 IP 地址，不包含其他个人信息
   - 数据仅用于在应用中显示位置信息

## 权限说明

### 位置权限（可选）

- **用途**：在菜单中显示您的大致地理位置
- **控制**：您可以在应用设置中关闭位置服务
- **数据处理**：位置数据仅在本地使用，不会上传

### 网络权限

- **用途**：查询外网 IP 地址和网络状态
- **必需性**：应用核心功能需要

## 数据安全

- 所有数据仅存储在您的设备上
- 应用不包含任何数据收集或分析代码
- 应用不使用任何第三方分析服务

## 开源透明

MyIP 是完全开源的项目：

- 源代码：https://github.com/pholex/MyIP
- 您可以审查所有代码以验证隐私承诺

## 您的权利

- **查看数据**：所有数据存储在 `~/Library/Application Support/MyIP/`
- **删除数据**：卸载应用即可删除所有本地数据
- **控制权限**：可在系统设置中管理应用权限

## 儿童隐私

MyIP 不针对 13 岁以下儿童，也不会故意收集儿童的个人信息。

## 政策变更

如果隐私政策有重大变更，我们会在 GitHub 仓库中更新此文档。
